二、算法题

题目

给你一份『词汇表』（字符串数组） words 和一张『字母表』（字符串） chars。 假如你可以用 chars 中的『字母』（字符）拼写出 words 中的某个『单词』（字符串），那么我们就认为你掌握了这个单词。

注意：每次拼写（指拼写词汇表中的一个单词）时，chars 中的每个字母都只能用一次。 返回词汇表 words 中你掌握的所有单词的 长度之和。

示例 1：

输入：words = ["cat","bt","hat","tree"], chars = "atach"
输出：6
解释：
可以形成字符串 "cat" 和 "hat"，所以答案是 3 + 3 = 6。

示例 2：

输入：words = ["hello","world","leetcode"], chars = "welldonehoneyr"
输出：10
解释：
可以形成字符串 "hello" 和 "world"，所以答案是 5 + 5 = 10。

解法一 （哈希Map）

思路

  • 显然，对于一个单词 word，只要其中的每个字母的数量都不大于 chars 中对应的字母的数量，那么就可以用 chars 中的字母拼写出 word。
  • 所以我们只需要用一个哈希表存储 chars 中每个字母的数量，再用一个哈希表存储 word 中每个字母的数量，最后将这两个哈希表的键值对逐一进行比较即可。

代码

var countCharacters = function(words, chars) {
    let map =  new Map()
    let cnt = 0
    for (let c of chars) {
        map.set(c,map.has(c)? map.get(c)+1:1)
    }
    for (let w of words) {
        if (check(w, new Map(map))) {
            cnt += w.length
        }
    }
    return cnt

};
function check(w, map) {
    for (let i of w) {
        if (map.has(i) && map.get(i)>0){
            map.set(i, map.get(i) - 1)
        }else{
            return false
        }
    }
    return true
}
